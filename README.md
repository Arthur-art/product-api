O Product API é um projeto de estudo e demonstração de arquitetura backend moderna com .NET 8, construído passo a passo com foco em escalabilidade, performance e boas práticas de engenharia de software.
A API segue princípios de Clean Architecture, injeção de dependência (DI) e separação de responsabilidades, preparando o terreno para integrações com Redis, Health Checks, resiliência (Polly) e Docker Compose.

O projeto foi desenvolvido para ser didático, modular e próximo de cenários reais de produção, permitindo fácil evolução para camadas adicionais como cache distribuído, mensageria, logs estruturados e métricas.
